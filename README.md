# desafio-dio-primeirositehtml

Criação de um site utilizando apenas HTML, com o propósito de treinar uso de tags e atributos de tags para melhor formatação do hipertexto.
## 💻 Resumo de Conteúdo
|   HTML   |   CSS(Interno)  |
|-----------|-----------|
|TAGS E ATRIBUTOS|ATRIBUTO "Style"|
## 💻 Código Base:
```
<DOCTYPE! Html>
<html>
		<head>
	<meta>
	<title></title>
		</head>
	<body>
	</body>
</html>
```
